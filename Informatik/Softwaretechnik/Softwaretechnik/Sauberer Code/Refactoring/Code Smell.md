## Beschreibung
Unter **Code Smell** versteht man übelriechenden Code, also Code, der [[Refactoring]] nötig hat.

Konkret bezeichnet es Code, der zwar funktioniert aber neu strukturiert werden muss.

## Verbreitete Smells
- Code Duplizierung
- Lange Methode
- Große Klasse (auch Gottobjekt)
- usw.
