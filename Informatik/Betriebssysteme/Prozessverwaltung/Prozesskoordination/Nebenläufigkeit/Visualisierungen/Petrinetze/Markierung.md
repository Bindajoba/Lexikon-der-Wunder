Eine Markierung ist so etwas wie ein Zustand eines [[Petrinetz]]

# Definition
EIne Markierung ist eine Abbildung $M: S \to \mathbb{N}$, durch die jeder Stelle eine Anzahl von Marken (Token) zugeordnet wird.
