# Definition
Sei $V$ ei [[Vektorraum]]. Eine Norm auf $V$ ist eine Abbildung $||\cdot||: V \to \R_0^+$, sodass für $x, y \in V, \lambda \in \R$ gilt
1) $||x|| = 0 \iff x = 0$
2) $||\lambda x|| = |\lambda|||x||$
3) $||x+y|| \leq ||x||+||y||$

Wir nennen einen Vektorraum mit einer Norm [[Normierter Vektorraum]]

# Beispiele
# Die $l_p$-Normen
Für $p \in \R, p \geq 1$ ist die $l_p$-Norm auf $\R^n$ definiert durch
$$||x||_p = (\sum_{i = 0}^n|x_i|^p)^{\frac{1}{p}}$$
Mit
- $p=1$ erhält man die Manhattan Norm
- $p = 2$ erhält man die Euklidische Norm
- $p = \infty$ oder streng $||x||_\infty = \underset{i = 0, ..., n}{\max}(|x_i|)$
