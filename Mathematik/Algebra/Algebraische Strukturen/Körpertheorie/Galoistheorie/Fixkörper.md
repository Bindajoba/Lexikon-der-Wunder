# Definition
Sei $L$ ein  [[Körper]] und $G$ eine [[Untergruppe]] von $Aut(L)$. Dann nennt man $L^G = \{\alpha \in L : \sigma(\alpha) = \alpha, \forall \sigma \in G\}$ den **Fixkörper** von $G$.
