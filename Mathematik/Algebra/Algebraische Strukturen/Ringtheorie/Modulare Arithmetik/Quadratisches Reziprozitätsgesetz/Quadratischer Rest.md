# Definition
Sei $p$ eine Primzahl und $a \in \Z$. Man nennt $a$ einen quadratischen Rest modulo $p$, wenn eine Zahl $c \in \Z$ mit $a \equiv c^2 \mod p$ existiert. Andernfalls spricht man von einem quadratischen Nichtrest.
